# Ícones do PWA

Para que o PWA funcione completamente, você precisa adicionar os seguintes ícones no diretório `static/`:

## Ícones Necessários

1. **icon-192x192.png** - Ícone de 192x192 pixels
2. **icon-512x512.png** - Ícone de 512x512 pixels

## Como Criar os Ícones

### Opção 1: Usando Ferramentas Online
- Acesse https://realfavicongenerator.net/ ou https://www.pwabuilder.com/imageGenerator
- Faça upload de uma imagem (recomendado: 512x512px ou maior)
- Baixe os ícones gerados e coloque-os neste diretório

### Opção 2: Criar Manualmente
- Use um editor de imagens (Photoshop, GIMP, Canva, etc.)
- Crie imagens quadradas (192x192 e 512x512 pixels)
- Use o logo do "Menino do Alho" ou um ícone relacionado
- Salve como PNG com fundo transparente (recomendado)
- Nomeie como `icon-192x192.png` e `icon-512x512.png`

### Opção 3: Gerar a partir de SVG
Se você tiver um logo em SVG:
- Use ferramentas como https://cloudconvert.com/svg-to-png
- Converta para PNG nas resoluções necessárias

## Nota Importante

Os ícones são essenciais para:
- Instalação do PWA no iPhone/iPad (Adicionar à Tela de Início)
- Instalação no Android (Add to Home Screen)
- Exibição na tela inicial do dispositivo
- Melhor experiência visual do aplicativo

**Sem os ícones, o PWA ainda funcionará, mas a experiência de instalação será limitada.**
